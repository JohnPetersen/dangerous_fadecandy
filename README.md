Dangerous Fadecandy
===================

Fadecandy playground with four [Neopixel matrices](https://www.adafruit.com/product/1487) and a [Danger Shield](https://www.sparkfun.com/products/11649) for control.

Display Modes
-------------
* Mouse controlled
* Bouncing ball
